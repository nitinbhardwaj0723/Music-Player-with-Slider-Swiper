# 🎵 Music Player with Slider (Swiper JS)

A modern and responsive Music Player UI built using **HTML, CSS, JavaScript**, and **Swiper.js**.  
This project features a smooth sliding interface for songs with interactive controls and clean UI design.

---

## 🚀 Features

- 🎧 Play / Pause functionality
- ⏭ Next / Previous track control
- 🎚 Smooth slider using Swiper.js
- 📱 Fully responsive design
- 🎨 Modern UI with animations
- 🔄 Real-time song switching
